IK Spider

Description

This is a simple setup for a spider-mech player, I like the way inverse kinematics allows for more biological looking movement.

Controls

Desktop Click to focus window, Arrow Keys to move, Mouse to aim. Press Enter to pause.

Mobile Use the touchscreen buttons, Tap to aim.
